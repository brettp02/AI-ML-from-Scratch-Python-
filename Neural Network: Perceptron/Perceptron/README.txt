Before attemptint to run make sure you have the libraries installed with the following command:
- pip install pandas && pip install numpy

To run the Perceptron without train/test split:
- navigate to Perceptron folder
- enter: python3 Perceptron.py ionosphere.data

To run the Perceptron with train/test split:
- navigate to Perceptron_split folder
- enter: python3 Perceptron.py ionosphere.data
